
# 🚀 CRM Dashboard (Full Stack Web App)

A modern **Customer Relationship Management (CRM) system** built with **React, Django, and MySQL**, featuring real-time updates, analytics dashboard, and a responsive SaaS-style UI.

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/crm-project.git
````

### 2. Backend setup

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### 3. Frontend setup

```bash
cd frontend
npm install
npm start
```

## 🔐 Environment Variables

### Backend `.env`

```
DB_NAME=your_db
DB_USER=root
DB_PASSWORD=your_password
DB_HOST=localhost

```

```
