# 🚀 CRM Dashboard (Full Stack Web App)

A mini **Client Lead Management system** built with **React, Django, and MySQL**, featuring real-time updates, analytics dashboard, and a responsive SaaS-style UI.

## ⚙️ Tech Stack

### Frontend
- React.js ⚛️
- Tailwind CSS 🎨
- Axios 🔗
- Chart.js 📊

### Backend
- Django 🐍
- Django REST Framework 🔥

### Database
- MySQL 🗄️

---

## ✨ Features

### 👥 Lead Management
- Add new leads
- Update lead status (New / Contacted / Converted)
- Delete leads
- Notes support

### 📊 Dashboard Analytics
- Total leads tracking
- Status breakdown cards
- Lead growth chart


### 🔍 UI/UX
- Modern SaaS-style dashboard
- Responsive design (mobile + desktop)
- Sidebar navigation with active state
- Clean glassmorphism UI

---

## 🧠 Key Highlights

- Full-stack CRUD system
- Secure environment variable setup
- REST API architecture
- Professional dashboard UI
